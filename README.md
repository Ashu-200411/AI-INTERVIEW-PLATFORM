# AI Interview Platform

An AI-powered interview preparation platform built with Next.js that helps users practice mock interviews, receive AI-generated feedback, and improve their interview performance.

## 🚀 Features

- Secure User Authentication
- AI-Powered Mock Interviews
- Technical & HR Interview Questions
- Real-time Video Interview Experience
- AI-Based Performance Feedback
- Interview History Dashboard
- Responsive UI for Desktop
- Fast and Secure Backend Integration

## 🛠️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- Shadcn UI

### Backend
- Next.js Server Actions
- Supabase

### Other Technologies
- Stream Video SDK
- Arcjet
- Vapi AI
- JavaScript

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ashu-200411/AI-INTERVIEW-PLATFORM.git
```

Navigate to the project:

```bash
cd AI-INTERVIEW-PLATFORM
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
